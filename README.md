QA Automation using Playwright

End-to-end (E2E) test automation project developed with Playwright, focused on QA best practices, project organization, and continuous learning.

This repository aims to:

- Demonstrate knowledge of E2E test automation
- Apply QA best practices
- Use the Page Object Model (POM)
- Create positive and negative test scenarios
- Serve as a professional QA portfolio project
- The project uses a public demo website to simulate real-world web application testing scenarios.
